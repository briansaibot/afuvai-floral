## General Rules to Check

### Typography
- [ ] No single words on their own line (widows)
- [ ] No text smaller than 12px on mobile (readability)
- [ ] Heading text uses proper line-height
- [ ] Paragraph text uses proper line-height (1.6+)

### Overflow & Wrapping
- [ ] No horizontal scrolling on mobile
- [ ] No buttons with text overflow
- [ ] No cards with text overflow
- [ ] Images don't overflow containers

### Responsive Breakpoints
- [ ] Mobile: < 640px (375px, 425px test)
- [ ] Tablet: 768px - 1024px (768px test)
- [ ] Desktop: > 1024px (1440px test)

### Edge Cases
- [ ] Very long product names (test with 30+ char names)
- [ ] Very long email addresses (test with long emails)
- [ ] Numbers without spaces (test with long order numbers)
- [ ] Multiple apostrophes in text (don't break styling)

---

## Tools & Methods

**Browser DevTools:**
1. Open Chrome DevTools (F12)
2. Click responsive design mode (Ctrl+Shift+M)
3. Set dimensions: 375px (mobile), 768px (tablet), 1440px (desktop)
4. Test text wrapping, overflow, button sizes

**Testing Pages:**
1. Visit each page in mobile (375px)
2. Scroll through entire page checking for overflow
3. Test form inputs (mobile keyboard might show)
4. Click buttons to ensure text stays inside
5. Resize viewport gradually from 375px to 1440px and watch for breaks

**What to Look For:**
- Text going off-screen (horizontal scroll)
- Text running into other elements (buttons, images)
- Awkward single-word line breaks
- Buttons too small to tap (< 48px tall)
- Input fields too small (< 44px tall)
- Images distorted or pixelated
- Uneven spacing due to text length
